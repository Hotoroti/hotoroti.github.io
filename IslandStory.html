Island

Here I made an Island in Blender

PolygonCount = 1079.

This is my first 3D object I made in Blender. I started to learn Blender because in my second your of college I learned how to make 3D games. So I tought this was a good time to learn how to model 3D. I followed the tutorial "Complete Blender Creator 3" on gamedev.tv. With this tutorial I learned the basics about blender.

First I made some simple houses. With this I learned a little about some edit tools, like loop cuts and other things. After that I made the light house. Here I put everything I learned to work. When I finished the light house I made the rocky base were the town is on. I learned the basics of how to use the sculpting tool.

After I finished making the base I learned various little things, like the decimal modifier, snapping and linked duplicates. And I went around and tried different things with it. Eventually the time came to make it look pretty, so I learned about materials and lighting.
